CURRENT PROBLEM:
- Status stays "PROCESSING" even when AI completes
- User must manually refresh to see "READY_TO_PUBLISH"

REQUIRED BEHAVIOR:
- Dashboard polls status every 5 seconds while processing
- Automatically updates to "READY_TO_PUBLISH" when AI finishes
- Shows "Preview" button as soon as ready

IMPLEMENTATION:
In Dashboard.tsx:
```typescript
useEffect(() => {
  if (status === 'processing') {
    const interval = setInterval(async () => {
      const response = await fetch('/api/profile/status');
      const data = await response.json();
      if (data.status === 'ready') {
        setStatus('ready');
        clearInterval(interval);
      }
    }, 5000); // Poll every 5 seconds
    
    return () => clearInterval(interval);
  }
}, [status]);
```

Add endpoint in routes.ts:
```typescript
app.get('/api/profile/status', async (req, res) => {
  const profile = await getProfileByUserId(req.user.id);
  res.json({ status: profile.status });
});
```
