FIX 2: FILTER OUT "NA" / "N/A" / "None" FROM ACHIEVEMENTS
Add this helper function:
```typescript
function cleanAchievements(achievements: string[]): string[] {
  return achievements
    .filter(a => a && a.trim().length > 0)
    .filter(a => !['na', 'n/a', 'none', 'nil', 'null'].includes(a.toLowerCase().trim()))
    .map(a => a.trim());
}
```

Apply to career history processing:
```typescript
const cleanedAchievements = cleanAchievements(role.achievements);
if (cleanedAchievements.length > 0) {
  // Only show achievements section if there are valid ones
  rolesData.push({
    ...role,
    achievements: cleanedAchievements
  });
}
```

FIX 3: REMOVE DOUBLE BULLETS IN ACHIEVEMENTS
Update achievement formatting:
```typescript
function formatAchievement(text: string): string {
  // Remove existing bullets/markers
  let cleaned = text.replace(/^[\s•\-\*]+/, '').trim();
  // Don't add bullet if already starts with number
  if (/^\d+\./.test(cleaned)) return cleaned;
  return cleaned;
}
```
